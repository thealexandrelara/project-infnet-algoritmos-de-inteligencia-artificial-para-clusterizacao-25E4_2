# Instalação e Execução do Projeto

Este projeto foi desenvolvido em Python 3.12.8 e com diversas dependências que podem ser vistas no arquivo `pyproject.toml` ou `requirements.txt`. Para facilitar a gestão de dependências e o ambiente virtual, é recomendado o uso do `uv`. Pode-se utilizar Anaconda ou `virtualenv` (isoladamente) se preferir, para isso realize a instalação de dependências utilizando o arquivo `requirements.txt`.

## Pré-requisitos

    • Python 3.12.8 instalado
    • uv instalado (instruções para instalação podem ser encontradas (aqui)[https://docs.astral.sh/uv/getting-started/installation/#installing-uv])

#### Instalação do projeto

Clone o repositório e instale as dependências:

```bash
git clone git@github.com:thealexandrelara/project-infnet-algoritmos-de-inteligencia-artificial-para-clusterizacao-25E4_2
```

```
cd project-infnet-project-infnet-ia-generativa-para-linguagem-large-language-model-25e2_3
```

#### Instale as dependências listadas em pyproject.toml

```bash
uv sync
```

#### Arquitetura do projeto

    |_ project.ipynb: Contém as respostas e implementações do projeto.
    |_ assets: Contém imagens do projeto.
    |_ data: Contém os dados baixados e utilizados na análise durante a execução do projeto.
